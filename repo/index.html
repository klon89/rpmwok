Electron RPMbuild

Electron packages for openSUSE, SLE, Fedora, CentOS

Add repository in openSUSE & SLE

sudo zypper ar -f https://electron-rpmbuild.guoyunhe.me/electron-rpmbuild.repo
sudo zypper ref

Add repository in Fedora & CentOS

sudo dnf config-manager --add-repo https://electron-rpmbuild.guoyunhe.me/electron-rpmbuild.repo
sudo dnf config-manager --set-enabled electron-rpmbuild
